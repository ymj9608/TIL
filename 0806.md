- 2차원 배열 리스트 컴프리헨션으로 입력하는 방법
  ```py
  N = int(input())
  arr1 = [list(map(int, input().split())) for _ in range(N)]
  """
  3
  1 2 3
  4 5 6
  7 8 9
  """
  print(arr1)

  N = int(input())

  456
  789
  """
  ```

- 델타를 활용한 2차원 배열 탐색
   - 인덱스 (i, j)인 칸의 상하좌우 칸 (ni, nj)
     ```py
     di[] ← [0, 1, 0, -1]
     dj[] ← [1, 0, -1, 0]
     for k : 0 → 3
        ni ← i + di[k]
        nj ← j + di[k]
     ```
